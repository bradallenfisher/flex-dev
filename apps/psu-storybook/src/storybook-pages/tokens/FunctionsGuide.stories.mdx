<h2 style={{ border: 'none', marginTop: '50px' }}>mergeSx Function</h2>

<p>Use the mergeSx if you need to merge multiple styles into one variable</p>

```jsx
import { mergeSx } from '..';

const YourComponent = () => {
  const baseSx = {
    color: 'nittanyNavy',
  };
  const borderSx = {
    borderWidth: 'xs',
  };
  const hoverSx = {
    '&:hover': { backgroundColor: 'linkLight' },
  };

  const componentSx = mergeSx(baseSx, borderSx, hoverSx);

  return <div sx={{ ...componentSx }}>...</div>;
};
```
