<h2 style={{ border: 'none', marginTop: '50px' }}>
  Accessing the spacing tokens
</h2>

<p>
  **Access any spacing token by using the sx prop and passing in the neccessary
  token key into margin, padding, scrollMargin, scrollPadding, top, right,
  bottom, left, gridGap, gridColumnGap, gridRowGap, gap, columnGap, or rowGap.
  These CSS properties will use values defined in the theme, when available.
</p>

```jsx
//for functional component
<Flex extraSx={{ mx: 5 }}>...</Flex>
<Flex extraSx={{ p: 2 }}>...</Flex>

//for base html element
<div sx={{ mx: 4 }}>This div is using margin 4 token</div>
<div sx={{ mb: 2 }}>This div is using margin 2 token</div>
<div sx={{ py: 2 }}>This div is using padding 2 token</div>
...
```
