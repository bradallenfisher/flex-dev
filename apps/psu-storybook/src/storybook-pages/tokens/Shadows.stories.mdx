<h2 style={{ border: 'none', marginTop: '50px' }}>
  Accessing the shadow tokens
</h2>

<p>
  **Access any shadow token by using the sx prop and passing in the neccessary
  token key into boxShadow or textShadow. These CSS properties will use values
  defined in the theme, when available.
</p>

```jsx
//for functional components
<Flex extraSx={{ boxShadow: 'md-2' }}>...</Flex>

//for html elements
<div sx={{ boxShadow: 'md-1' }}>Box Shadow</div>
<h1 sx={{ textShadow: 'sm-1' }}>Text shadow</h1>
```
