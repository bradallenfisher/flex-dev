<h2 style={{ border: 'none', marginTop: '50px' }}>
  Accessing the gradient tokens
</h2>

<p>
  **Access any gradient token by using importing the theme and adding the
  gradient as a backgroundImage
</p>

```jsx
import { gradients } from '..'

//for functional components
<Flex extraSx={{ backgroundImage: () => `${gradients['hero']}`}}>...</Flex>

//for base html elements
<div sx={{ backgroundImage: () => `${gradients['hero']}`}}>...</div>
<div sx={{ backgroundImage: () => `${gradients['lightBox']}`}}>...</div>
<div sx={{ backgroundImage: () => `${gradients['powerFacts']}`}}>...</div>
```
