<h2 style={{ border: 'none', marginTop: '50px' }}>
  Accessing the radius tokens
</h2>

<p>
  **Access any radius token by using the sx prop and passing in the neccessary
  token key into borderRadius, borderTopRightRadius, borderTopLeftRadius,
  borderBottomRightRadius, borderBottomLeftRadius. These CSS properties will use
  values defined in the theme, when available.
</p>

```jsx
//for functional components
<Flex extraSx={{borderRadius: 'xs'}}>...</Flex>

//for html elements
<div sx={{ borderRadius: 'xs' }}>4px Border Radius</div>
<div sx={{ borderTopRightRadius: 'sm' }}>8px Border Top Right Radius</div>
<div sx={{ borderBottomLeftRadius: 'md' }}>12px Border Bottom Left Radius</div>
```
