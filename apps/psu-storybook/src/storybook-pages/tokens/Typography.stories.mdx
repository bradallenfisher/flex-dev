<h1 >Typography</h1>

<p sx={{ color: 'link' }}>
  Typography variants automatically include responsive values for fontSize,
  lineHeight, letterSpacing, and fontFamily.
  <br />
  Using variants is <b>always</b> recommended.
</p>

<h4>Example:</h4>

```jsx
// variant={20} includes fontFamily, fontWeight, letterSpacing, lineHeight values at each breakpoint
<Body variant={20}>Body</Body>
```

</div>

<h2>How to tell which variant to use?</h2>

<p>
  Variant key name corresponds with the Typography font size at the desktop
  size. This is true for all typography components (Body, Heading, DisplayHeading)

<br />

So if the design has a font size for a Heading of 32px at the desktop screen size, then the correct variant is 32

</p>

```jsx
// for DisplayHeading with 64px fontSize at desktop
<DisplayHeading variant={64}>Responsive 64 variant for DisplayHeading</DisplayHeading>

// for Heading with 32px fontSize at desktop
<Heading variant={32}>Responsive 32 variant for Heading</Heading>

// for Heading with 20px fontSize at desktop
<Body variant={20}>Responsive 20 variant for Body</Body>


```

<h2>Available variants for each component</h2>

<p>
  Use the corresponding key that matches the desktop fontSize in the design.
</p>
<br />
<SbTypographyComponent />

<h2>Typography Tags</h2>

<p>
There is a default tag for each functional typography component (Body, Heading, DisplayHeading)
<br />

If needed this can be overwritten with the tag prop.

</p>

```jsx
<DisplayHeading tag="h2">h2 DisplayHeading</DisplayHeading>
<Heading tag="h3">h3 Heading</Heading>
```

<h2>Overwriting properties</h2>

<p>
  If possible, its <b>always</b> recommended to use variants, however when the
  design has a slight difference from the variant, you can use extraSx to pass
  in the needed token.
</p>

```jsx
// overwrite default variant fontWeight of bold on the Heading to the medium token
<Heading variant={32} extraSx={{ fontWeight: 'medium' }}>
  Medium fontWeight Heading
</Heading>

// overwrite default variant lineHeight to 120 token
<Heading variant={32} extraSx={{ lineHeight: '120' }}>
  120% lineHeight Heading
</Heading>
```
