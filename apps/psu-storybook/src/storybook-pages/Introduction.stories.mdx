# Getting started with psu-flex

<span className="body">
  Psu-flex is a multi-tenant platform solution that offers an on brand and
  streamlined approach to get a Penn State site up and running quickly. The
  platform leverages CMS along with Netlify's static site
  generation platform to build a performant site that can be customized to match
  each departments needs.
</span>
